1. 복사생성자의 호출
   1) 생성된 객체로 새 객체를 생성시
   2) 함수의 전달인수가 객체일때
   3) 함수의 반환값이 객체일때
2. 복사생성자
   1) 복사생성자를 삽입하지 않으면 compiler가 삽입
      == default 복사생성자
   2) default 복사생성자는 구체적인 일을 한다.
       cf) default 생성자, default 소멸자는 일을 안함
   3) 복사생성자 형태
     클래스명(const 클래스명  레퍼런스객체변수){
        생성자함수가 하는 일만 함!!
    }
  4) 
     i)깊은 복사 :
     멤버변수가 동적할당된 경우는 반드시 복사생성자를 삽입
     ii)얕은 복사 :
     멤버변수가 동적할당되었지만 복사생성자가 없을때
3. const멤버변수
    : 모든 멤버메소드 안에서 변경불가

   1) 생성자함수에서 const멤버변수의 초기치는?  
        생성자함수의 멤버이니셜라이징기법으로!! 
    생성자함수의 헤더:멤버변수1(매개변수1),멤버변수2(매개변수2){
       생성자함수의 body
    }
4. const멤버함수
   1) const멤버함수안에서는 모든 멤버변수의 변경불가!
   2) const멤버함수안에서 호출된 함수도 const함수여야 함
   3) const멤버함수의 반환값은 포인터형이어서는 안된다.
